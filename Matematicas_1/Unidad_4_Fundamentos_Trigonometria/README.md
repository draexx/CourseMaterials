## Clases

### Clase 1: Introducción a la Trigonometría y Funciones Básicas

**Duración:** 1 hora 20 minutos

**Objetivos:**
- Comprender los conceptos básicos de trigonometría (40%)
- Identificar las funciones trigonométricas fundamentales (40%)
- Aplicar en problemas prácticos (20%)

**Desarrollo:**
1. **Introducción (10 min)**
   - Definición de trigonometría
   - Aplicaciones en la vida real
   - Ejemplo práctico: Navegación y cartografía

2. **Funciones trigonométricas básicas (30 min)**
   - **Definiciones:**
     \[
     \sin \theta = \frac{\text{cateto opuesto}}{\text{hipotenusa}}, \quad \cos \theta = \frac{\text{cateto adyacente}}{\text{hipotenusa}}, \quad \tan \theta = \frac{\sin \theta}{\cos \theta}
     \]
   - **Círculo unitario:**
     \[
     \sin^2 \theta + \cos^2 \theta = 1
     \]
   - **Aplicación:** Cálculo de ángulos y distancias

3. **Actividad práctica (20 min)**
   - Ejercicios de cálculo de funciones
   - Problemas de triángulos rectángulos
   - Ejemplos prácticos

4. **Aplicación en la vida real (20 min)**
   - Ejemplos de navegación
   - Cálculo de alturas de edificios
   - Problemas de topografía

5. **Cierre (15 min)**
   - Resumen de conceptos
   - Asignación: Problemas de aplicación
   - Presentación de proyecto

**Materiales:**
- Calculadora científica
- Hojas de cálculo
- Diagramas de triángulos
- Modelos de triángulos rectángulos

**Recursos Adicionales:**
- Teóricos:
  - [Khan Academy - Trigonometría](https://es.khanacademy.org/math/trigonometry)
  - [Wikipedia - Trigonometría](https://es.wikipedia.org/wiki/Trigonometr%C3%ADa)
- Prácticos:
  - [Calculadora de Trigonometría](https://www.calculadora-trigonometria.com/)
  - [Ejercicios de Trigonometría](https://www.emathhelp.net/es/calculators/algebra-2/)
- Ejercicios:
  - [Paul's Online Math Notes](https://tutorial.math.lamar.edu/)
  - [GeoGebra - Trigonometría](https://www.geogebra.org/)

**Evaluación:**
- Participación en ejercicios prácticos (30%)
- Tareas semanales con problemas aplicados (30%)
- Proyecto final: Análisis de aplicaciones (40%)

**Proyecto Integrador:**
- Análisis de problemas de navegación
- Cálculo de alturas y distancias
- Aplicación en topografía

### Clase 2: Funciones Inversas y Propiedades

**Duración:** 1 hora 20 minutos

**Objetivos:**
- Dominar las funciones trigonométricas inversas (40%)
- Aplicar propiedades de las funciones trigonométricas (30%)
- Resolver problemas prácticos (30%)

**Desarrollo:**
1. **Introducción (10 min)**
   - Definición de funciones inversas
   - Ejemplos prácticos

2. **Funciones trigonométricas inversas (30 min)**
   - **Definiciones:**
     \[
     \arcsin, \arccos, \arctan
     \]
   - **Dominio y rango:**
     \[
     \arcsin(x): [-1, 1] \rightarrow \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]
     \]
   - **Aplicación:** Cálculo de ángulos

3. **Propiedades de las funciones (30 min)**
   - **Periodicidad:**
     \[
     \sin(\theta + 2\pi) = \sin \theta
     \]
   - **Simetría:**
     \[
     \sin(-\theta) = -\sin \theta, \quad \cos(-\theta) = \cos \theta
     \]
   - **Identidades fundamentales:**
     \[
     \sin^2 \theta + \cos^2 \theta = 1
     \]

4. **Ejercicios y Aplicaciones (20 min)**
   - Problemas de despeje de ángulos
   - Cálculo de distancias
   - Ejemplos prácticos

5. **Cierre (15 min)**
   - Resumen de propiedades
   - Asignación: Problemas de aplicación
   - Presentación de proyecto

**Materiales:**
- Calculadora científica
- Tabla de identidades trigonométricas
- Diagramas de funciones
- Problemas de aplicación

**Recursos Adicionales:**

  - [Wikipedia - Funciones Trigonométricas](https://es.wikipedia.org/wiki/Funci%C3%B3n_trigonom%C3%A9trica)

  - [Ejercicios de Funciones](https://www.emathhelp.net/es/calculators/algebra-2/)
- Ejercicios:
  - [Paul's Online Math Notes](https://tutorial.math.lamar.edu/)
  - [GeoGebra - Ejercicios](https://www.geogebra.org/)

**Evaluación:**

- Tareas semanales con problemas aplicados (30%)

### Clase 3: Triángulos Rectángulos y Teorema de Pitágoras

**Duración:** 1 hora 20 minutos

**Objetivos:**
- Resolver triángulos rectángulos (40%)
- Aplicar el teorema de Pitágoras (30%)
- Desarrollar habilidades de resolución (30%)

**Desarrollo:**
1. **Introducción (10 min)**
   - Características de triángulos rectángulos
   - Ejemplos prácticos

2. **Razones trigonométricas (30 min)**
   - **Relaciones:**
     \[
     \sin \theta = \frac{\text{cateto opuesto}}{\text{hipotenusa}}
     \]
   - **Cálculo de lados y ángulos:**
     \[
     \text{hipotenusa} = \sqrt{\text{cateto opuesto}^2 + \text{cateto adyacente}^2}
     \]
   - **Aplicación:** Cálculo de distancias

3. **Teorema de Pitágoras (25 min)**
   - **Demostración:**
     \[
     a^2 + b^2 = c^2
     \]
   - **Casos especiales:**
     \[
     3-4-5, \quad 5-12-13
     \]
   - **Aplicación:** Cálculo de diagonales

4. **Actividad Práctica (20 min)**
   - Problemas de triángulos
   - Cálculo de alturas
   - Ejemplos prácticos

5. **Cierre (15 min)**
   - Resumen de métodos
   - Asignación: Problemas de aplicación
   - Presentación de proyecto

**Materiales:**
- Calculadora científica

### Clase 4: Leyes de Senos y Cosenos

**Duración:** 1 hora 20 minutos

**Objetivos:**
- Aplicar la ley de senos (40%)
- Aplicar la ley de cosenos (30%)
- Resolver problemas prácticos (30%)

**Desarrollo:**
1. **Introducción (10 min)**
   - Triángulos oblicuángulos
   - Ejemplos prácticos

2. **Ley de Senos (30 min)**
   - **Fórmula:**
     \[
     \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}
     \]
   - **Casos de aplicación:**
     - AAL (Ángulo-Angulo-Lado)
     - LAL (Lado-Angulo-Lado)
   - **Aplicación:** Navegación

3. **Ley de Cosenos (30 min)**
   - **Fórmula:**
     \[
     c^2 = a^2 + b^2 - 2ab \cos C
     \]
   - **Uso en triángulos:**
     - Lados y ángulos dados
     - Problemas de distancias
   - **Aplicación:** Topografía

4. **Ejercicios Integradores (20 min)**
   - Problemas combinados
   - Cálculo de distancias
   - Ejemplos prácticos

5. **Cierre (15 min)**
   - Resumen de leyes
   - Asignación: Problemas de aplicación
   - Presentación de proyecto final

**Materiales:**
